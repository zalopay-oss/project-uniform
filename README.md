## Giới thiệu
Sự gắn kết giữa các thành viên với nhau do nhiều yếu tố tạo thành. Khi đi cùng nhau, mang trên mình chiếc áo team sẽ dễ dàng nhận ra hơn rất nhiều so với khi không mặc.

## Design T-shirt
